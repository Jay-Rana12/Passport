# Global Visa Solutions Website

Professional Visa & Passport Consultancy Website template.

## Setup

1.  No installation required. This is a static HTML/CSS/JS website.
2.  Open `index.html` in your web browser to view the site.

## Structure

-   `css/style.css`: Main stylesheet containing variables and component styles.
-   `js/script.js`: Handle navbar interactions, scrolling animations, and other dynamic effects.
-   `images/`: Contains generated assets (Logo, Hero background, etc.).
-   `*.html`: core pages (Home, About, Visa, etc.).

## Customization

-   To change colors, edit the root variables in `css/style.css`.
-   To update content, edit the respective HTML files.
-   To add more images, place them in the `images` folder and reference them in the HTML.
